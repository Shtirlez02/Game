Игра напоминает "Пятнашки", только  вместо цифр - картинка.
Перед началом игрок видит целую картинку, которую нужно запомнить.
Затем по нажатию на Enter картинка разбивается на ровные квадраты,
которые расставляются случайным образом.
Нужно собрать картинку, передвигая элементы рисунка, кликая по ним мышкой.
Элемент картинки передвигается на свободное рядом место.
Игра продолжается до тех пор, пока не будет собран рисунок.
